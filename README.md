一个`Vue.js 3` UI 组件库

## 安装

```shell
./install
```

<details>
  <summary>Tip</summary>

直接运行`pnpm install`会导致内部命令链接错误
</details>

## 构建

<details>
  <summary>组件库</summary>

```shell
pnpm run packages:build
```

</details>

<details>
  <summary>文档</summary>

```shell
pnpm run docs:build
```

</details>

## 测试

<details>
  <summary>组件库</summary>

```shell
pnpm run packages:test
```

</details>
