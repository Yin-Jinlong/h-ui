# Badge 标记

## 颜色

按钮的颜色。可以是已经定义的颜色（如`primary`），
也可以是有效的`css`颜色值（如`red` `#39e` `var(--my-color-var)`）。

[.colors]

[.count]

## 溢出显示

[.overflow]

## 内容

当内容为`string`时，`min` `max`将会忽略。为空（`''`）时不显示。

[.content]

## 大小

可以通过修改`font-size`来调整大小。

`badge`的`font-size`为`0.5em`

[.size]
